Developing Embedded System Prototypes

This is my website world where I share all about my experiences

Components List

- Elegoo Uno R3
- 16x2 Liquid Crystal Display (LCD) (Qty: 4)
- Micro pushh buttons (Qty: 5
- 5v realy
- Potentiometer
- Green Run LED

This design and project was made with the intent to start an electric motor control panel at user's desire. It has since developed into something that i would like to call a smart starting computer that can not only enable a start command for a control panel but any machine or device that requires a circuit completion to start.

Later on in the project, the design then featured autonomous starting options that the user would be able adjust by navigating through the LCD screen. The starter will also feature an RTC (real time clock) that will be help the device to start anything attached to it at any given time the user desires. The project it still in its prototyping stage and is not yet complete as more research is being completed on the real time clock portion of the project.
